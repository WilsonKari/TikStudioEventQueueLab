# TikStudioEventQueueLab

Laboratorio en C++ creado con Visual Studio y CMake.

## Requisitos

- Visual Studio con las herramientas de desarrollo de C++.
- CMake 3.10 o posterior.
- Ninja.

## Compilar

Selecciona uno de los presets definidos en `CMakePresets.json`, por ejemplo `x64-debug`, desde Visual Studio o CMake.
